# node-ts-template

> node 服务端环境 typescript 模板文件


```js
# build
npm run build

# 
npm run start

npm run test
npm run test:coverage

```

## 目录说明

Project
|---build                 //编译后的js文件目录
|---src                   //ts文件目录
|  |----index.ts          // 入口文件
|  |---types                
|  |     |---type.d.ts         // type 定义文件
|  |----__test__          // 测试用例目录
|---index.js              // 原项目的index
|---libs                  // 原项目的libs
|---coverage              // 测试覆盖率结果
|---tsconfig.json         // ts配置
|---jest.config.js        // jest 测试框架的配置
|---tslint.json      
|---package.json
